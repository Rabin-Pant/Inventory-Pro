Project Overview
InventoryPro is a full-stack inventory management system built with Flask and React. It provides role-based access control for administrators and customers, enabling efficient product management, order processing, and automated invoice generation.

Purpose
Streamline inventory management for small to medium businesses

Provide separate interfaces for administrators and customers

Automate order processing and invoice generation

Offer real-time stock tracking and analytics

Target Users
Business Owners: Manage products, track orders, analyze revenue

Customers: Browse products, place orders, download invoices

Store Managers: Update inventory, process orders, manage categories
